# 🏠 Cozy Room — Three.js Journey Exercise

An interactive 3D scene "Cozy Room". This project demonstrates 3D modeling integration, custom lighting, and interactive elements using the **React Three Fiber** ecosystem.

## 📂 Project Structure

- `public/models/` — Contains the 3D assets (`.glb`) and baked textures.
- `src/components/` — Core React-Three-Fiber components (e.g., `CozyRoom.jsx`).
- `src/hooks/` — Custom hooks for asset management and logic.
- `src/materials/` — Custom shader materials and Three.js material configurations.

## 🚀 Tech Stack

- **React 18** + **Vite** — Fast frontend tooling and UI management.
- **Three.js** — Core WebGL library.
- **@react-three/fiber** — React bridge for Three.js.
- **@react-three/drei** — Functional helpers for cameras, staging, and shaders.
- **Leva** — Real-time GUI for tweaking scene parameters.
- **R3F-Perf** — Performance monitoring tool for 3D frames.
